// - - - - PROCEDURE INSTALLATION LOCALE - - - - //

Installer un serveur local pour DEV : WAMP, MAMP...

Déplacer le dossier cserrurerie à la racine du serveur (htdocs/ sur MAMP, www/ sur WAMP je crois ?)

Configurer la BDD : 
	nom: cserrurerie
	user: root
	password: root
	hôte: localhost
	prefixe: cs_
